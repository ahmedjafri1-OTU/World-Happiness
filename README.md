# 🌍 World Happiness Analysis using Clustering
This project analyzes global happiness indicators using unsupervised machine learning techniques to identify patterns and relationships between socio-economic factors and happiness scores.
### 📊 Data Exploration
Data analysis and preprocessing are performed using:
* Pandas
* NumPy
* Seaborn
  
Correlation analysis is used to understand relationships between key variables such as GDP per capita, social support, life expectancy, and perceptions of corruption.

### 🤖 Unsupervised Learning
Clustering is performed using the K-Means algorithm implemented through Scikit-learn.
Key steps include:
* 📍 Applying K-Means clustering to group countries with similar happiness indicators
* 📉 Using the Elbow Method to determine the optimal number of clusters
* 📊 Visualizing relationships between clusters and happiness scores
